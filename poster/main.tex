\begin{exampleblock}{Introduction}
Full fine-tuning large language models is expensive. \lora{} reduces the cost by training low-rank adapter updates while freezing the base model.\shortcite{hu2022lora}

\compactheading{DoRA}
\begin{itemize}
  \setlength{\itemsep}{0.05ex}
  \setlength{\parskip}{0pt}
  \item \dora{} decomposes pretrained weights into \textbf{magnitude} and \textbf{direction}.\shortcite{liu2024dora}
  \item Direction receives a \lora{}-style low-rank update; magnitude is learned separately.
\end{itemize}

\compactheading{Project goal}
\begin{itemize}
  \setlength{\itemsep}{0.05ex}
  \setlength{\parskip}{0pt}
  \item Does \dora{} improve commonsense accuracy over a same-scope \lora{} baseline?
  \item Reproduce the LLaMA2-7B trend and test attention-only, MLP-only, and full-scope adapters.
\end{itemize}
\end{exampleblock}

\begin{block}{Conclusion \& Future Work}
\vspace{-2ex}
\heading{Takeaways}
\vspace{-2ex}
\begin{itemize}
  \setlength{\itemsep}{0.15ex}
  \setlength{\parskip}{0pt}
  \item Full-scope \dora{} improved macro-average accuracy by \textbf{+1.79 points}.
  \item The reproduction supports the qualitative value of weight-decomposed low-rank adaptation.
  \item The ablation adds a caution: applying \dora{} narrowly can remove the benefit.
\end{itemize}

\vspace{-2ex}
\heading{Next steps}
\vspace{-2ex}
\begin{itemize}
  \setlength{\itemsep}{0.15ex}
  \setlength{\parskip}{0pt}
  \item Train on the full \texttt{commonsense\_170k} dataset.
  \item Repeat all six conditions across multiple random seeds.
  \item Add parameter-count, GPU-memory, and wall-clock cost comparisons.
\end{itemize}
