\begin{exampleblock}{Introduction}
Full fine-tuning large language models is expensive. \lora{} made fine-tuning cheaper by training low-rank updates while freezing the base model. \dora{} makes this process even cheaper.

\compactheading{DoRA (Weight-Decomposed Low-Rank Adaptation)}
\begin{itemize}
  \setlength{\itemsep}{0.05ex}
  \setlength{\parskip}{0pt}
  \item \dora{} decomposes pretrained weights into \textbf{magnitude} and \textbf{direction}.
  \item Direction receives a \lora{}-style low-rank update; magnitude is learned separately.
\end{itemize}

\compactheading{Project goal}
\begin{itemize}
  \setlength{\itemsep}{0.05ex}
  \setlength{\parskip}{0pt}
  \item Maximize fine-tuning performance with \dora{} while using fewer parameters.
  \item Test whether \dora{} beats \lora{} at the same rank and where it helps most: attention, MLP, or full-scope adapters.
\end{itemize}
\end{exampleblock}

\begin{block}{Conclusion \& Future Work}
\vspace{-2ex}
\heading{Takeaways}
\vspace{-2ex}
\begin{itemize}
  \setlength{\itemsep}{0.15ex}
  \setlength{\parskip}{0pt}
  \item Full-scope \dora{} improved macro-average accuracy by \textbf{+1.79 points} while using fewer trainable parameters.
  \item \lora{} is still strong at higher ranks, but performs worse than \dora{}.
  \item \dora{} worked best when applied fully across the transformer.
\end{itemize}

\vspace{-2ex}
\heading{Potential Next steps}
\vspace{-2ex}
\begin{itemize}
  \setlength{\itemsep}{0.15ex}
  \setlength{\parskip}{0pt}
  \item Train on the full \texttt{commonsense\_170k} dataset to address small dataset worries.
  \item Repeat the LoRA and DoRA variants across multiple random seeds to verify consistency.
  \item Check more rank comparisons at full scale to verify the trend across more ranks.
\end{itemize}
